# 个人主页 模板

这是为仓库 `ZhaoYufei0914/ZhaoYufei0914.github.io` 准备的一个简单个人主页模板，包含：

- `index.html` — 页面主体
- `css/styles.css` — 样式文件
- `assets/avatar.jpg` — 请自行添加头像文件（或修改 HTML 中的路径）

如何使用
1. 将 `index.html` 放到仓库根目录（或 `docs/`）。
2. 添加 `css/styles.css` 到 `css/` 目录。
3. 把你的头像放到 `assets/avatar.jpg`（或修改 `index.html` 中的 `img` 地址）。
4. 修改 `index.html` 中的文本（姓名、简介、邮箱、项目链接等）。
5. 推送到 GitHub 仓库。

部署到 GitHub Pages
- 打开仓库的 Settings → Pages。
- 选择 Branch：`main`（或 `gh-pages`），并选择 Root（如果文件在根目录）或 `/docs`（如果放在 docs/）。
- 保存并等待几分钟，你的主页将出现在 `https://ZhaoYufei0914.github.io/`。

自定义建议
- 替换头像与简介文本。
- 在 Projects 区块添加真实项目链接与简短说明。
- 如果需要自定义域名，可添加 `CNAME` 文件并在 GitHub Pages 中设置自定义域名。

如果你想让我帮你直接把这些文件推到仓库（我可以生成提交内容示例或帮你写出要推送的完整文件内容），告诉我你希望放在根目录还是 `docs/`，我会继续帮助你。
